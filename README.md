# Cinematic Xenon Presentation

A fullscreen interactive presentation website built with Next.js, React, Tailwind CSS, Framer Motion, and Three.js.

## Local Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## GitHub Pages

This project is configured for static export with GitHub Pages.

1. Push the repo to GitHub.
2. In GitHub, go to `Settings > Pages`.
3. Set `Source` to `GitHub Actions`.
4. Push to `main` or run the `Deploy to GitHub Pages` workflow manually.

The workflow automatically sets the correct base path for normal project pages, such as `/your-repo-name/`. For a user or organization Pages site named `username.github.io`, it builds with no base path.
